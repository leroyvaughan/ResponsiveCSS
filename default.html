Responsive CSS Demo

This is a demonstration page for Responsive CSS. If you look at the page border while resizing the browser window, or on different types of clients (ie. desktop: wide, narrow, tablet, mobile), you'll notice that the it changes colors as the screen size varies. I use this so that I know what screen size I'm dealing with when looking at page elements that are out of place on smaller screen size. Typically I will code for the desktop environment, then scale the site down to a max-width of 290px.

There are three stylesheets referenced by this page:

  • Styles.css: the main stylesheet
  • MediaQueries.css: this is where the responsive adjustments are made in the CSS.
  • MediaQuery_colors.css: this is the one used solely for the page border color changes

For the NAV MENU button, I modified the CSS for this particular view port size (using max-width css query). The click handler and other associated javascript for the navigation menu can be found in the "main.js" file. The JavaScript utilizes an html element (#cssViewPort) that is positioned off-screen.

In the MediaQueries.css file, whenever a new viewport size is encountered, that particular html element has it's "left" property changed to match that of the media query max-width. This "left" property is set so that in the JavaScript when the getViewPortSize() function is called on page load and on window.resize() it can be set to a JavaScript variable.

The JavaScript variable (viewPort) is utilized so that the code will know when to apply the click handler to the NAV MENU, and also when to remove it.
